This repository holds jinja2 templates for [nrx](https://github.com/netreplica/nrx) tool. It was cloned from https://github.com/netreplica/templates and switched to be hosted under the current origin.

Prefer exporting into cyjs format first, and them importing from cyjs to export via the templates. Only re-export from netbox when data in previously exported cyjs file is expected to be outdated.
